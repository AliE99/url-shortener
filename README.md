#URL_SHORTENER
make links shorter

####Run
```
dotnet run
```
####Test
```
dotnet test
```
####EndPoint
For Post and shortener your URL :
http://localhost:5000/urls
```json
{
	"longUrl":"http://alibaba.ir"
}
```
For Get and redirect :
http://localhost:5000/redirect/"shortUrl"
